# ShoppyGlobe Backend API

#  Project Description
This is an e-commerce backend project built using Node.js, Express.js and MongoDB.

This project includes:
- User Registration & Login (JWT Authentication)
- Product API
- Add to Cart
- Update Cart
- Delete Cart
- MongoDB Database integration
- ThunderClient testing

# Technologies Used
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- ThunderClient
- GitHub

#  API Routes

#  Auth APIs
POST /api/auth/register  
POST /api/auth/login  

# Product APIs
GET /api/products  
GET /api/products/:id  

# Cart APIs (Protected)
POST /api/cart  
PUT /api/cart/:id  
DELETE /api/cart/:id  

#  Testing
All APIs tested using ThunderClient.
